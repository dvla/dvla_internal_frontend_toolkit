DVLA internal frontend toolkit

This frontend toolkit is for internal DVLA services. It contains the styles, templates, components and patterns that developers need to create them.

Using the toolkit

Depending on your development needs, you have a choice of two packages:

  • dvla-internal-frontend-toolkit
  • dvla_internal_frontend_toolkit

The source is code is available at:

  • dvla_internal_frontend_toolkit [GitHub]

Prototyping

The frontend toolkit is also used in the DVLA internal prototype kit. You can download the prototype kit from Github:

  • DVLA internal prototype kit [Node/Express app]

For advice on how to use the prototype kit, there is a prototype guidance section on this site.
